BEM (Block, Element, Modifier) là một phương pháp luận đặt tên lớp CSS để giúp các nhà phát triển web viết mã CSS có cấu trúc và có thể tái sử dụng một cách dễ dàng. BEM tạo ra một quy ước đặt tên nhất quán và rõ ràng, giúp việc duy trì và mở rộng mã CSS trở nên dễ dàng hơn.
